此版本开始完成用户注册功能。

上一个版本中，我们已经让HttpRequest支持了不同的请求方式
（带参数的和不带参数的）
因此，本版本来完成处理注册业务。

首先我们要修改ClientHandler处理请求环节。
因为已经不能再用HttpRequest的uri作为请求路径进行判断了（
因为uri可能含有参数部分），我们要改用为requestURI的值，
这个值是表示请求部分的。
