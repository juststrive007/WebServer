本版本开始整理请求的内容
思想：
一个请求的内容东西由很多，因此我们设计一个类，名为
httpRequest，并且将请求中各个信息以属性定义。
然后每当我们解析一个客户端发送过来的请求后都实例化一个
HTTPRequest，并将信息赋值到对应属性。这样将来处理请求时
我们可以获取对应属性来得到请求中相关信息。

实现：
1.首先创建一个包：com.webserver.http
  这个包用来保持所有有关HTTP协议的内容
2.在http包中定义类：HTTPRequest
  该类的每个实例将来用于表示客户端发送过来的一个请求
  内容
3。在ClientHandler中实例化HttpRequest完成第一步处理
  工作:解析请求
  而解析请求的实际操作由httpRequest的构造方法完成，
  即：实例化HttpReq的同时将客户端发送过来的请求解析完毕
  ，使得当前HttpRequest对象就是表示该请求的。
